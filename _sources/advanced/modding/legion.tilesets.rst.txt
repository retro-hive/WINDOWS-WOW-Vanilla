Legion Tilesets
===============

`Legion Tilesets`_ is a backport of the tilesets from ``Legion`` expansion
into the ``Vanilla`` client.

Download
--------

* Official `Link`_
* `Archive.org`_

Installation
------------

+ Rename the file to ``patch-4.MPQ``
+ Copy the file ``patch-4.MPQ`` into the ``Data`` directory of your WoW installation.
